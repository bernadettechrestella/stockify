# STOCKIFY

**Stockify** is a fullstack Inventory Management Web App
for managing products, categories, stock and pricing.

## Features

- User Registration & Login (JWT)
- Product CRUD (name, price, stock, category)
- Category CRUD
- Search, Filter, Pagination
- Dashboard UI

## Tech Stack

**Frontend:** React (Tailwind or MUI)  
**Backend:** Express.js + MySQL (Pure SQL)  
**Database:** MySQL
